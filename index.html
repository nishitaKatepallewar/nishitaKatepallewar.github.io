Core Features

Built for
game pipelines.

Every check tuned to game-ready engine constraints — Unity, Unreal, Godot.

01
Topology Analysis

Detects ngons, non-manifold geometry, overlapping faces, zero-area polygons, and triangle density violations.

02
UV Validation

Checks for UV overlaps, out-of-bounds islands, flipped UVs, texel density consistency, and lightmap compliance.

03
AI Scene Reading

Reads your asset's role and adjusts tolerance thresholds accordingly.

04
LOD Compliance

Validates LOD chains — poly ratios, naming conventions, transition flags, and missing LOD detection.

05
Naming Standards

Enforces studio naming conventions across meshes, materials, textures, and bones.

06
Export Reports

One-click PDF or JSON reports with ranked issues, screenshots, and fix suggestions.

Workflow

Four steps
to clean assets.

Step 01
Install & Configure

Install the .zip in Blender preferences. Set your engine target and naming rules once per project.

Step 02
Select Assets

Select one object, a collection, or the entire scene. Vexa works at any scope.

Step 03
Run Vexa

Hit the Vexa panel button. The AI engine analyses geometry, UVs, materials, hierarchy in seconds.

Step 04
Fix & Export

Review the ranked issue list. Apply one-click auto-fixes, then export a clean QC report.
